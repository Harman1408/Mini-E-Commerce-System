# Mini E-Commerce System

## Description
A Mini E-Commerce System built using MySQL.  
This project manages products, customers, orders, and order items, demonstrating basic database operations and inventory management.

---

## Tables
The database contains the following tables:

- **Products** – Stores product details (ID, name, category, price, stock).  
- **Customers** – Stores customer details (ID, name, email, phone, address).  
- **Orders** – Stores order details (ID, customer ID, date, total amount).  
- **Order_Items** – Stores items in each order (quantity, price, product).

---

## Features
- Add and view products  
- Add and view customers  
- Place orders and track order items  
- Update stock automatically when orders are placed  

---

## Setup Instructions
1. Install MySQL and open your MySQL client.  
2. Create the database and tables by running your **CREATE TABLE commands**.  
3. Insert sample data using your **INSERT INTO commands**.  
4. You can then use MySQL queries to test orders, sales, and stock management.
